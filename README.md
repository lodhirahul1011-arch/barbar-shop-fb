
# BarberFlow Clean MERN Architecture

## Stack
- React.js Frontend
- Node.js Backend
- Express
- MongoDB
- JWT Authentication

## Included
- Login
- Signup
- JWT auth middleware
- MongoDB User model
- Separate frontend/backend folders
- Booking module placeholder

## Run Backend
cd backend
npm install
npm run dev

## Run Frontend
cd frontend
npm install
npm start
